Classpublic class OSMFSparkPlayer
InheritanceOSMFSparkPlayer spark.components.supportClasses.SkinnableComponent

OSMFSparkPlayer is a Spark component that wraps the OSMF framework in a Spark style component API to easily skin an OSMF based media player.
