Classpublic class PlaylistRenderer
InheritancePlaylistRenderer almerblank.flex.spark.components.SkinnableItemRenderer
Implements IPlaylistRenderer

PlaylistRenderer is the default item renderer object used with the OSMFPlaylist. You can create your own item renderer by making sure you implement IDataRenderer, IItemRenderer and IPlaylistRenderer.
