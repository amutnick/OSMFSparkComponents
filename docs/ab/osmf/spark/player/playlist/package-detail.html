 OSMFPlaylist OSMFPlaylist is a Spark component used in conjunction with an OSMFSparkPlayer instance to create a playlist of OSMF supported media to display.
 PlaylistRenderer PlaylistRenderer is the default item renderer object used with the OSMFPlaylist.
 URLType URLType is used with OSMFPlaylist to create URLResource objects for media items fed into the playlist.
